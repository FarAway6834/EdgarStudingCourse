# 정 : 필기에 대한 의문

## 문자셋 Ler, 문자열셋 Str정의
```text/plain korean style lojban mathmatics
Ler := {x | "le me’o x. cu lerfu"}
Str := {x | x ∈ Lerⁿ⁽ˣ⁾ (n(t) ≡ |t|)}
```
변명 : 한국식 측도의 구문론적 정의 `(n(t) ≡ |t|)`를 Str에서만 적용시켜 `ⁿ⁽ˣ⁾`를 쓰는건, 유니코드에서 기술할려면 다른 방법이 없기 때문.
